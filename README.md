# GanSystem Weed Detection

AI-powered weed detection system developed for smart agriculture.

## Project Overview

This project was developed to automatically identify weeds within bean farms using computer vision and machine learning techniques.

## Dataset

* 213 agricultural images
* 2334 annotated objects
* Classes:

  * Bean
  * Weed

## Data Preparation

Preprocessing:

* Auto orientation
* Contrast enhancement
* Image resizing (640x640)

Augmentation:

* Horizontal flipping
* Rotation
* Blur
* Motion blur
* Noise injection
* Zoom augmentation

## Model Performance

* mAP@50: 65.1%
* Precision: 42.2%
* Recall: 81.8%
* F1 Score: 55.7%
